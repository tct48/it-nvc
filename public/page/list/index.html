เขียน HTML ให้ได้ตัวอย่างผลลัพพธ์ตามด้านบน

โดยมีรายละเอียดของสีดังนี้

1. สี background ของแต่ละ li คือ rgba(0, 0, 0, 0.904);
2. แท็ก h4 ที่อยู่ใน li อันแรกเป็นสี deepink ส่วนอันอื่น ๆ เป็นสี deepskyblue
3. เมื่อเอาเมาส์ไปชี้ที่ li แต่ละอันจะมีรายละเอียดดังนี้
3.1 สีพื้นหลังเปลี่ยนเป็นสี rgba(0, 0, 0, 0.853);
3.2 รูปเมาส์เปลี่ยนเป็นรูปชี้นิ้ว ( pointer )
4. โดย li แต่ละอันจะอยู่ใน div container ที่มีขนาด 350 px;

ตัวอย่างการเขียน HTML และ CSS (ใบ้)

1. การเขียน HTML (100%)

2. การเขียน CSS ช่วยแค่ (20%)

การบ้านฝึกนี้ทำแล้วได้อะไร

1. ฝึกการใช้ ul li

2. ฝึกการเรียกใช้ class

3. ฝึกการเรียกใช้ font จากภายนอก

4. ฝึกใช้เพื่อจะแยกแยะ margin กับ padding ให้เก่งมากขึ้น
